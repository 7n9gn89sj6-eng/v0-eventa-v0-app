# Glossary System

The glossary provides two levels of explanation:

1. **Lightbox** (2–4 sentences) — Short explanations that appear in popups
2. **Deeper Notes** (300–600 words) — Longer context pages accessible via "Notes" links

## Structure

Each glossary term has:
- `[term]-lightbox.md` — Short explanation for lightbox
- `[term]-notes.md` — Deeper notes page (optional, only if needed)

## Usage in Content

- Link to lightbox: Use inline links that trigger a lightbox popup
- Link to notes: Use "Notes" links that open a separate page
