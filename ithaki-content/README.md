# Ithaki Group Tour — Content System

This directory contains the content architecture for ithakigrouptour.com.

## Structure

- `pages/` — Main page content (ready to paste into Wix)
- `glossary/` — Glossary entries (lightbox and deeper notes)
- `linking-rules.md` — Guidelines for when and how to link

## Working Method

Content is written in plain markdown/text format for easy copying into Wix Classic Editor. Each file is self-contained and can be pasted directly.

## Core Principles

- Main text reads cleanly on its own
- Links are optional side doors, not footnotes
- No sales language, no hype
- Calm, precise, unpretentious tone
- Historically grounded

## Design Preferences

- **Font:** Helvetica Neue (to be set in Wix design settings)
